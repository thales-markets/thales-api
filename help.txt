docker build -f Dockerfile.orders -t thalesmarket/orders  .

### OPEN REDIS CLI ON SERVER ###
- Access SERVER
- Run: docker ps
- Copy "CONTAINER ID" of redis:alpine
- Run: docker exec -it "CONTAINER ID" redis-cli
